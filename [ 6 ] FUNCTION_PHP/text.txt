function adalah potongan program atau baris - baris kode
yang kita buat untuk mempermudah kita pada saat memprogram.
kode program tadi bisa kita kasih nama, lalu bisa kita panggil berulang-ulang.

- built-in-function --> fungsi yang dimiliki PHP
- user-defined-function -->. fungsi yang kita buat sendiri

# [+] built-in-function
ada beberapa fungsi yang digunakan :
- Date/Time :
1. time()
2. date() --> ini tuh memanggil function yang telah ada di PHP.
3. mktime()
4. strtotime()

- String :
1. strlen() --> untuk menghitung panjang sebuah String
2. strcmp() --> untuk membandingkan sebuah String
3. explode() --> memecah sebuah String menjadi array
4. htmlspecialchars()
